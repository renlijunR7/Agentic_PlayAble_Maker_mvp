# Agentic PlayAble Maker MVP V2 原型

这是产品经理提供的 V2 静态前端原型，已清理旧版与无关截图/上传素材，方便提交到 GitHub。

## 运行方式

直接打开 `index.html`，或在本目录启动一个静态服务：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## 说明

- 当前项目为静态原型，无需安装依赖。
- 页面通过 CDN 加载 React、ReactDOM 与 Babel，因此离线环境下可能无法正常渲染。
- 入口文件：`index.html`
