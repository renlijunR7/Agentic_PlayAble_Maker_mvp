F-01

入口与意图捕获

把 KOL 的一句话变成结构化的 Brief
P0 · MUST
四类入口 必须支持：首页"新建" / 模板中心"套用" / 信息中心通知跳转 / 品牌简报响应
意图大文本框 必须支持自然语言输入，⌘ + Enter 提交，提交按钮在 prompt 为空时禁用
可选挂载：资料上传 / 选分身 / 选模板。挂载越多，Forge 反问越少
会话上下文：分配 Session ID，挂积分钱包，记录所有挂载资产
语音输入：可用麦克风口述意图（v1.1 引入）
验收 · ACCEPTANCE
用户输入 5-200 字意图 + 任意挂载，能进入 Phase B
所有 4 类入口均可跳进工作台，带上下文参数
空意图时提交按钮置灰
不允许在没有意图的情况下生成 v1
F-02

三栏工作区 · 共创界面

左对话 / 中预览 / 右变体 · 三栏强联动
P0 · MUST
左栏 · Forge 对话：消息流（KOL 与 Forge 两类气泡）、输入框、积分消耗提示、分身/资料/模板挂件按钮
中栏 · 实时预览：iPhone 设备框、3 段循环（intro / playing / reward）、底部三指标（完成率 / 分享率 / 每局成本）
右栏 · 变体看板：v1 / v1.a / v1.b / v1.c 卡片，状态徽章（当前 / A/B 中 / 草稿）、复制 / 预览 / 删除
联动规则：左栏发消息 → 中栏 5-12s 内刷新预览 → 右栏自动新建变体
设备切换：预览支持 iPhone / Android / 横屏 / 竖屏
验收 · ACCEPTANCE
三栏在 1440 宽下完整可见，无横向滚动
预览 3 段循环可自动播放，时长 2.4s + 3.8s + 2.4s
每次对话提交，预览刷新 ≤ 12 秒
变体看板最多并列 6 个，超出滚动
F-03

对话引擎 · Forge Brain

理解意图 / 反问 / 生成 / 调整
P0 · MUST
简报反问：首次提交后必须问 ≤ 3 个补全问题（时长 / 目标 / 受众）
多轮上下文：保留整段会话历史，最长 50 轮，超出后自动总结
差异化变体：在 v1 生成时同时给出 3-4 个变体，每个标注差异点和预估提升
积分计费：每轮对话消耗 38 积分，余额不足时禁止发送
情绪标签：Forge 气泡支持表情（happy / thinking / wink / concerned）
引用提及：消息内可以 @某个变体名或分身名，自动建立链接
验收 · ACCEPTANCE
反问回答后 ≤ 12 秒出 v1
同一意图重跑 3 次，变体差异显著（结构 / 文案 / 视觉至少其一）
积分扣减事务化，失败时回滚
不允许 Forge 在没有反问的情况下直接出 v1
F-04

变体管理 · Variants

命名 / 分叉 / 合并 / A/B 跑量
P1 · MUST
变体树：v1 → v1.a / v1.b 二级分叉，最多 3 级
状态机：草稿 / 当前 / A/B 中 / 已归档 四种状态
A/B 编排：选两个变体并行发布，自动按 50/50 流量分配，14 天后给冠军
变体复制：一键复制变体作为新分支起点
变体合并：把两个变体的局部（开场+奖励）合成第三个
验收 · ACCEPTANCE
同会话可创建 ≥ 8 个变体
A/B 跑量 14 天后自动出冠军报告
每个变体独立保存预估指标
F-05

资产挂载 · Assets Mount

把分身 / 品牌 / 模板挂进 Playable
P1 · MUST
分身挂载：从分身工厂选一个，自动注入语气、形象、台本
品牌挂载：从品牌简报库选简报，自动注入 logo、奖池预算、合约条款
模板挂载：从模板中心选一个，{标的}/{奖池}/{主题} 槽位自动暴露
资产冲突检测：如挂的分身语气和品牌简报语气矛盾，标黄提示
验收 · ACCEPTANCE
挂载后 ≤ 2 秒预览反映变化
资产能 原子化撤销（不影响其他挂载）
冲突告警必须可关闭
F-06

深度体检 & 发布

模拟 1000 玩家 + 合规扫描 + 三种发布通道
P0 · MUST
模拟跑量：发布前自动跑 1,000 次模拟玩家，覆盖地区/语言/设备分布
风险扫描：识别风险词（涉赌 / 违禁 / 仿冒），标黄但不阻断
平台合规：检查每个分发平台的特定约束（TikTok 时长 / IG 比例 / X 字符），出修改建议
三种通道：8 平台一键 / Agent 世界 9 节点 / 导出工程包
健康度报告：完成率、分享率、风险、合规四项打分 → 绿/黄/红
验收 · ACCEPTANCE
体检全流程 ≤ 30 秒
红灯禁止发布，必须修改后才能继续
8 平台分发 5 秒内全部排队成功
导出包 .zip 含 Playable + 简报 PDF + 合约
F-07

自进化闭环

数据回流 + 主动提建议 + 抽模板
P2 · SHOULD
数据回流：每 1 小时拉一次实时指标（完成率/分享/收益）回工作台
主动通知：发布后 24h / 168h 各推送一次 A/B 建议，回到信息中心
学习成熟度：在左栏 Forge Agent 旁显示成熟度百分比（34% → 持续上升）
模板抽取：跑通的会话可一键"存为模板"，自动识别可变槽位
验收 · ACCEPTANCE
回流数据 ≤ 1 小时延迟
24h 后的建议命中率 ≥ 60%（被采纳率）
模板抽取后立即出现在模板中心

类目指标说明
性能v1 生成 ≤ 12s · p95意图提交到首次预览渲染完成的端到端时间
性能每轮迭代 ≤ 8s · p95对话提交到预览刷新
性能体检 ≤ 30s · p951000 次模拟 + 风险 + 合规全跑完
可用性99.5% 月度可用工作台核心 API 月度 SLO，每次掉线影响 ≤ 50 KOL

国际化5 种语言界面中英印葡西，Forge 对话支持自动语言切换

可达性键盘可操作所有按钮和输入框支持 Tab 导航、屏读器朗读
数据会话 1 年保留完整会话和变体保留 365 天，之后归档冷存
观测全链路埋点从入口到发布的每个步骤打 trace，用于诊断"为什么会话卡住"
